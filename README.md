# 🏫 Campus Connect

A secure, institute-only collaboration platform for students to share study resources, join academic groups, post updates, and receive real-time notifications — built with the MERN stack.

> 🔐 Verified access • 📚 Course-specific resources • 👥 Group collaboration • 🔔 Real-time alerts



## ✨ Features

- **Secure Registration**: Sign up with college email (`@*.edu.in`) or college ID (e.g., `BCA2025103`)
- **Email Verification**: Ensures only valid students join
- **User Profiles**: Editable bio, semester, course, batch + avatar upload
- **Community Feed**: Create text/image posts (campus-wide or group-specific)
- **Academic Groups**: Create/join clubs with member management
- **Resource Sharing**: Upload PDFs, PPTs, DOCX files tagged by subject, course, and semester
- **Real-Time Notifications**: Instant alerts for likes, comments, and invites (via Socket.IO)
- **Smart Filtering**: Search posts/resources by course, semester, or group

## 🛠️ Tech Stack

- **Frontend**: React 18, Redux Toolkit, React Router v6, Tailwind CSS, Framer Motion  
- **Backend**: Node.js, Express, MongoDB (Mongoose), Zod validation  
- **Auth**: JWT + bcrypt password hashing  
- **Storage**: Supabase (for file uploads)  
- **Realtime**: Socket.IO  
- **Security**: Dotenvx (encrypted secrets), institute-level data isolation
